# ぺたりん

**見ているページのドメインごとに、付箋をぺたり。**
画面の端からそっと顔を出す WEB 付箋 Chrome 拡張機能です。

## これは何？

「ぺたりん」は、いま見ている Web ページのドメインに紐づいた付箋を貼れる拡張機能です。
付箋は画面の端に貼り付き、ふだんは小さく格納されてページの邪魔をしません。気になったときだけ開いてメモできます。

- **ドメインごとに保存** — `example.com` に貼った付箋は、次に同じドメインを開いたときにまた現れます
- **端に吸い付く** — 付箋は画面の端（右・左・上・下）に磁石のように貼り付きます
- **ふだんは小さく** — 格納中は先頭の数文字だけを表示。開くとフルサイズに展開
- **ポップアップで一望** — どのドメインに貼ったか忘れても、拡張機能アイコンから全部まとめて探せます

## 使い方

1. ツールバーの「ぺたりん」アイコンの近く、画面の端に黄色い **「＋」** の付箋が出ています
2. **「＋」をクリック** すると、いま見ているドメインに新しい付箋が展開状態で生まれます。そのまま書き込めます（最大 2 行）
3. 付箋の **背（帯の部分）をドラッグ** すると、端に沿って移動できます（右配置なら上下だけ、上配置なら左右だけ）
4. 背を **クリック** すると展開／格納が切り替わります。格納中は先頭 3〜4 文字だけが見えます
5. 展開中は **色を選択**（デフォルトは黄色）でき、**削除ボタン** も出ます
6. 付箋をたくさん開いたら、**画面の外側をクリック / Esc キー / 「まとめてとじる」ボタン** でいっぺんに格納できます

## ポップアップ（管理画面）

ツールバーのアイコンをクリックすると管理画面が開きます。

- 保存中のすべての付箋を **ドメインごと** に一覧表示（いま開いているドメインは「今ここ」表示で先頭に）
- 付箋やドメインを **検索**
- 付箋カードをクリックすると **そのドメインを開く**（どこに貼ったか忘れても飛べます）
- **配置** を右・左・上・下から選択
- **格納中は半透明にする** をオン／オフ（マウスを乗せるとはっきり表示されます）
- **ページ上に付箋を表示** をオフにすると、このポップアップだけで管理できます

## インストール（開発版）

1. このリポジトリを取得し、アイコンを生成します
   ```bash
   pnpm install
   pnpm run generate-icons
   ```
2. Chrome で `chrome://extensions` を開き、右上の **デベロッパーモード** をオン
3. **「パッケージ化されていない拡張機能を読み込む」** から、このフォルダを選択

配布用 ZIP は `./zip.sh`（macOS / Linux）または `./zip.ps1`（Windows）で作成できます。
