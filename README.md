# ぺたりん

**見ているページのドメインごとに、付箋をぺたり。**
画面の端からそっと顔を出す WEB 付箋 Chrome 拡張機能です。

## これは何？

「ぺたりん」は、いま見ている Web ページのドメインに紐づいた付箋を貼れる拡張機能です。
付箋は画面の端に貼り付き、ふだんは小さく格納されてページの邪魔をしません。気になったときだけ開いてメモできます。

- **ドメインごとに保存** — `example.com` に貼った付箋は、次に同じドメインを開いたときにまた現れます
- **端に吸い付く** — 付箋は画面の端（右・左・上・下）に磁石のように貼り付きます
- **ふだんは小さく** — 格納中はタブに絵文字アイコンを 1 つ表示。開くと普通の付箋のような箱に展開して複数行メモできます
- **ぜんぶまとめて管理** — どこに貼ったか忘れても、「付箋デスク」で全サイトの付箋を一覧・検索できます
- **データは端末の中だけ** — 付箋は自分のブラウザに保存され、外部に送信されません（同期を ON にしたときだけブラウザ標準の同期を使います）

## インストール

1. [Chrome ウェブストアのぺたりんのページ](https://chromewebstore.google.com/detail/dgdffcdhlghngidilklaobaaedmacdph)を開きます
2. **「Chrome に追加」** をクリック
3. ツールバーに「ぺたりん」のアイコンが増えれば準備完了です

> Microsoft Edge など、Chrome ウェブストアに対応した Chromium 系ブラウザでもそのまま追加できます。

## 使い方

1. Web ページを開くと、画面の端に黄色い **「＋」** の付箋が出ています
2. **「＋」をクリック** すると、いま見ているドメインに新しい付箋が展開状態で生まれます。そのまま複数行を自由に書き込めます
3. 付箋の **背（帯の部分）をドラッグ** すると、端に沿って移動できます（右配置なら上下だけ、上配置なら左右だけ）
4. 背を **クリック** すると展開／格納が切り替わります。格納中はタブに絵文字アイコンが 1 つ見えます
5. 展開中は **色や絵文字アイコンを選択**（デフォルトは黄色）でき、右上の **×（閉じる）** と下部の **🗑（削除）** ボタンが使えます
6. 付箋をたくさん開いたら、**画面の外側をクリック / Esc キー / 「まとめてとじる」ボタン** でいっぺんに格納できます

## ポップアップ

ツールバーの「ぺたりん」アイコンをクリックすると、いま見ているサイト向けの設定パネルが開きます。

- **どの端に貼る？** — 付箋を貼る位置を右・左・上・下から選択
- **格納中は半透明にする** — オンにすると、格納中の付箋がうっすら透けます（マウスを乗せるとはっきり表示）
- **ページ上に付箋を表示** — オフにすると、ページには出さずポップアップと付箋デスクだけで管理できます
- **検索** — 付箋やドメインをさっと検索。付箋カードをクリックすると **そのドメインを開きます**
- **付箋デスクを開く** — 全サイトの付箋をまとめて整理する画面（次の項）へ

## 付箋デスク

ポップアップの **「付箋デスクを開く」** から、全サイトの付箋をまとめて管理する画面が別タブで開きます。

- **サイトごとに一覧** — 左の索引でサイトを選ぶと、そのサイトの付箋だけを表示
- **並べ替え** — 新しい順・古い順・サイト順・色・文字数で整理
- **検索** — 付箋やサイトを横断して検索
- **まとめて操作** — 付箋を複数選んで、まとめて剥がす（削除）
- **書き出し／読み込み** — 全付箋を JSON ファイルにバックアップ・復元（読み込みは新しい付箋を追加するだけで、既存はそのまま）
- **複数PCで同期** — 同期パネルの入口（次の項）

## 複数PCで同期（任意・既定オフ）

複数の端末で付箋を共有したいときは、付箋デスクの **「同期」** から有効にできます。

- 同期は **ブラウザ標準のアカウント同期**（Google / Firefox アカウント）を通じて行われます。**ぺたりん開発者のサーバーには送信されず、開発者はデータにアクセスできません**
- 一度も ON にしていなければ、付箋はすべて端末内に保存され、外部送信は一切ありません
- 同期するもの（付箋だけ／見た目の設定も）や対象サイトを選べます。容量の目安はパネルのゲージで確認できます
- 同期の ON / OFF はその端末だけに適用されます（他の端末を勝手に有効化しません）
- 同期できるブラウザはアカウントの種類ごとに分かれます（Chrome・Edge・Firefox をまたいだ同期はできません）
